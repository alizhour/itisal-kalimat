Information Collection and Use

The Application may collect limited data when used, such as:

  • Device Internet Protocol (IP) address
  • Operating system
  • App usage statistics (e.g., pages visited, time spent)
  • Game progress and achievements (when using cloud save features)
  • Account information when connecting through Google (Android) or Apple ID (iOS) for progress synchronization

This data is used to improve the app, ensure a safe user experience, and provide cloud save functionality for game progress.

No personally identifiable information is knowingly collected from children under 13.

Cloud Save and Account Integration

The Application uses Firebase services to provide cloud save functionality. Users can connect their Google account (on Android) or Apple ID (on iOS) to save and synchronize their game progress across devices. This connection is optional and users can play without linking an account.

When you choose to connect your account, we may collect:

  • Basic profile information (username/display name)
  • Game progress data and achievements
  • Device identifiers for synchronization purposes
